Batch 2

Given an array called munchkins as such:

var munchkins = [ {shirt: "red, name: "frisky"}, {shirt: "blue, name: "shorty"}, {shirt: "purple, name: "misty"}, {shirt: "green, name: "evil"}, {shirt: "purple, name: "mighty"}];

Filter out all of the objects containing purple shirts

Batch 3: Reduce!

Given an array of munchkin numbers as such:

var munchkins = [5,7,2,10,1];

Use Array.reduce so that add all of the munchkin numbers are added together.

Batch 4: Map!

Given an array of munchkin numbers as such:

var munchkins = [5,7,2,10,1];

Use Array.map so that each munchkin number is now the square of its original number [25,49,4,100,1]
